# Booksy

Sistema de venta de libros online con React y Spring Boot.

## Tecnologías

**Frontend:**
- React 18.2.0
- React Router 6.20.0
- React Bootstrap 2.9.0
- Axios 1.6.2

**Backend:**
- Java 17
- Spring Boot 3.2.0
- Spring Security + JWT
- H2 Database
- Maven

## Instalación

**Backend:**
```bash
cd backend
mvn spring-boot:run
```

**Frontend:**
```bash
cd frontend-react
npm install
npm start
```

El backend corre en `http://localhost:8080` y el frontend en `http://localhost:3000`.
